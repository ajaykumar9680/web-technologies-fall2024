about react?

React is a popular JavaScript library for building user interfaces, particularly for single-page applications.
It allows developers to build large applications that can update and render efficiently in response to data changes

"The key feature of React is its virtual DOM, which improves performance by updating only the parts of the page that have changed"

Features of React

  1. Component Based Architecture
  2. OneWay Data Binding
  3. Virtual DOM
